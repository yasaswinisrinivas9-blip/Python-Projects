# Calculator

## Description

A command-line calculator built with Python that performs basic arithmetic operations using functions and a dictionary-based operation lookup.

The calculator allows users to perform multiple calculations continuously without restarting the program.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Continuous calculations
- Start a new calculation without exiting
- Function-based operation mapping

---

## How It Works

1. Enter the first number.
2. Choose an arithmetic operation.
3. Enter the second number.
4. The calculator performs the selected operation.
5. Continue calculating using the previous result or start a new calculation.

---

## Example

```text
What is the first number?: 20

+
-
*
/

Pick an operation: *

What is the second number?: 5

20 * 5 = 100

Type 'y' to continue calculating with 100, or type 'n' to start a new calculation:
```
