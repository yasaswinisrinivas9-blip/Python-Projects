# Hangman

## Description

A command-line implementation of the classic Hangman game built using Python.

The player attempts to guess a hidden word one letter at a time while avoiding running out of lives.

---

## Features

- Random word selection
- ASCII art for each life
- Input validation
- Win/Lose detection
- Letter tracking
- Multiple lives
